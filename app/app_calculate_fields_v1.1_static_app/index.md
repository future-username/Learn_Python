# Документация по калькулятору

Этот документ содержит ссылки на все диаграммы, описывающие структуру и работу приложения калькулятора.

## Содержание

1. [Диаграмма классов](class_diagram.md) - описывает структуру классов приложения и их взаимосвязи
2. [Диаграмма последовательности](sequence_diagram.md) - показывает последовательность взаимодействия между компонентами
3. [Диаграмма компонентов](component_diagram.md) - отображает компоненты системы и их взаимодействие
4. [Диаграмма состояний](state_diagram.md) - описывает состояния приложения и переходы между ними

## Краткое описание приложения

Приложение представляет собой простой калькулятор, реализованный с использованием библиотеки Tkinter. Калькулятор позволяет выполнять операцию сложения двух чисел.

Основные особенности:
- Валидация вводимых значений
- Нормализация ввода (замена запятых на точки, удаление пробелов)
- Обработка ошибок при некорректном вводе

## Как использовать диаграммы

- **Диаграмма классов** поможет понять структуру кода и взаимосвязи между классами
- **Диаграмма последовательности** покажет порядок выполнения операций
- **Диаграмма компонентов** даст представление о компонентах системы
- **Диаграмма состояний** поможет понять логику работы приложения

Эти диаграммы содержат достаточно информации для воссоздания кода приложения без дополнительных вопросов.
